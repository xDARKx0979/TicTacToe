About Run 3

Run 3 is an exciting endless runner game where you navigate through space tunnels. Use your skills to avoid falling into space as you jump from platform to platform.

Controls:

  • Use the arrow keys to move and turn
  • Press the spacebar to jump
  • Press Up arrow to flip gravity in certain levels

Run 3 features multiple characters with unique abilities, various tunnel types, and an engaging storyline across different game modes.
